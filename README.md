# Rainfall Prediction in India

# Overview
This project focuses on predicting annual rainfall in different regions of India using machine learning techniques. The dataset contains historical rainfall data across multiple states and years.

# Objective
To build regression models that can predict annual rainfall based on monthly and seasonal rainfall data.

# Dataset
- Source: GitHub  
- Dataset: Rainfall in India (1901–2015)  
- Link: https://github.com/sudheerachary/rainfall-prediction/blob/master/data/rainfall_in_india_1901-2015.csv  

# Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

# Features Used
- Subdivision (State/Region)  
- Year  
- Monthly rainfall (Jan–Dec)  
- Seasonal rainfall (Jan-Feb, Mar-May, Jun-Sep, Oct-Dec)  

# Project Workflow
- Data Loading  
- Data Cleaning  
- Exploratory Data Analysis (EDA)  
- Feature Selection  
- Model Building  
- Model Evaluation  

# Machine Learning Models Used
- Linear Regression  
- Decision Tree Regressor  
- K-Nearest Neighbors Regressor  
- Random Forest Regressor  

# Results
The regression models were trained to predict annual rainfall. Random Forest Regressor and Decision Tree models provided better accuracy compared to other models.
